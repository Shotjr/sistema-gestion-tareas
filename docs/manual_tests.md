# Pruebas Manuales - Sistema de Gestión de Tareas

## Caso 1: Registro de usuario
- Objetivo: Validar que un usuario pueda registrarse correctamente.
- Pasos:
  1. Ingresar a la página de registro.
  2. Completar los campos obligatorios.
  3. Hacer clic en "Registrar".
- Datos de prueba: usuario@mail.com / contraseña123
- Resultado esperado: El sistema muestra mensaje de confirmación y permite iniciar sesión.

## Caso 2: Creación de tarea
- Objetivo: Validar que un usuario pueda crear una tarea.
- Pasos:
  1. Iniciar sesión.
  2. Ir a "Nueva tarea".
  3. Ingresar título y descripción.
  4. Guardar.
- Resultado esperado: La tarea aparece en la lista de pendientes.
